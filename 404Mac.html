Beyond 404 For Mac User

Are You a Target User?

This project is designed for people who play Elden Ring on Windows Computer with Steam. If you use Mac, I assume you're a developer who knows some UNIX scritps.

Follow the Steps

1. Copy your save file ER0000.sl2 into your Mac Downloads folder

2. Make Sure websocket.js is also in your Downloads folder and reserve port 8080 for save file rendering

3. Use Bash or Fish and execute the following command, the script is pretty understandable

cd ~/Downloads; mkdir -p temp_server; cp websocket.js ./temp_server; cp ER0000.sl2 ./temp_server; cd temp_server; npm init -y; npm install ws; node websocket.js
